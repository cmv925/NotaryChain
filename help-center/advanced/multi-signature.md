# Multi-Signature

**Joint deeds, corporate filings, partnership agreements — multiple signers, one coordinated ceremony.**

## When to use it
- Joint deeds (married couples)
- Corporate resolutions requiring multiple officers
- Partnership and operating agreements
- Beneficiary acknowledgements
- Multi-party affidavits

## How it works
- All signers receive scheduled invitations
- Each verifies their own identity independently
- The ceremony begins when all signers are present (or in sequence if pre-configured)
- Single Hedera anchor covers the whole document
