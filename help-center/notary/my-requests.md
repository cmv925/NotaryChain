# My Requests

**Your accepted, in-flight notarization work. One-click Start or Join.**

## What it is
Your personal queue of paying assignments. Each row shows the signer, document type, status, and a Start / Join Session button.

## How to use it
1. Open **My Requests** tab.
2. Optionally run [AI Co-pilot](./ai-copilot.md) before starting.
3. Click **Start Session** → run the ceremony → click **Complete**.

## Pro tip
Run Co-pilot *before* clicking Start. It pre-loads risk scores, signer red flags, and a 5-item ceremony checklist.
