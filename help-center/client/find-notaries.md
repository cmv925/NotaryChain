## FAQ
**Why do I have to pick by state?** Notarization is a state-licensed act. A Florida notary can serve any signer in any state, but only via FL-compliant RON. We auto-filter so you only see notaries whose commission is valid for your transaction.

**Can I keep using the same notary?** Absolutely — favorite them in your profile and they'll appear first next time.

**Is there a service fee?** Notary fees are set by each notary. Platform fees are bundled transparently — no surprise charges at checkout.
