# Users Management

**Find any user in 2 seconds. View, enable, disable. Every action audit-logged.**

## How to use it
- **Search** by email, name, or ID
- **Filter** by role (user / notary / admin)
- **Click** any user to see their full profile + Enable/Disable buttons

## Pro tip
Disable accounts, don't delete them. Status changes are audit-logged; deletions aren't. Compliance and customer-support workflows depend on this distinction.
