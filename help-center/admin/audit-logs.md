# Audit Logs

**Every privileged action ever taken on the platform — searchable, exportable, subpoena-ready.**

## What's logged
- All admin actions (user enable/disable, notary approve/reject)
- Configuration changes (fees, alert thresholds, policies)
- Authentication events (logins, MFA, password resets)
- API key generation and revocation
- Document access by support staff

## Pro tip
When the auditor asks "who changed the fee structure on June 14?", you find it in 5 seconds.
