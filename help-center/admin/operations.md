# Operations Center

**Your incident-response war room. Hedera, S3, Stripe, MongoDB — all monitored live.**

## Five panels
1. **Hedera health** — mainnet latency, transaction throughput
2. **Storage analytics** — S3 usage, cost trends
3. **Payments** — Stripe authorisation rate, refund rate
4. **MongoDB** — query latency, connection pool
5. **Service Health & Incidents** — outage history with one-click PDF export

## Pro tip
Configure **Alert Settings** to ping you on Slack when Hedera latency exceeds 2 seconds. Catches outages before customers complain.
