# Oracle Watchlist

**The only RON platform that watches state legislatures in real time. Email + Slack alerts on every rule change.**

## What it is
A regulatory oracle that monitors notary and RON regulations across all US states. When a state updates its rules, the watchlist surfaces a flag — within minutes of publication, not quarters later.

## How to configure
1. Click any watchlist row to edit
2. Pick **jurisdictions** to monitor (FL, TX, NY, CA, VA, all)
3. Set **severity floor** (Low / Medium / High)
4. Add notification channels — emails, Slack webhook
5. Save

## Pro tip
Set high-severity Slack alerts for **FL + TX**. Those two states drive 60% of US RON volume — you can't afford to miss a rule update.
